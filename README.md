# Restaurant-Management-System
## Description
This is a simple Restaurant Management System written in C++.  
It uses CSV files to store customer data.  
The program allows you to add, update, delete, and display customers using a menu-driven interface.

## Features
Add Customer
 Show Customers
 Update Customer
 Delete Customer
 CSV file handling for data persistance
 Modular code using header and cpp files

## Files Included
 main.cpp : Main program with menu  
 customer.h : Customer module header file  
 customer.cpp : Customer module implementation file  
 Customers.csv : Stores customer data 

 ## How to Run
1. Clone or download the repository.  
2. Make sure all files (main.cpp, customer.h, customer.cpp, Customers.csv) are in the same folder.  
3. Compile main.cpp using a C++ compiler (GCC, OnlineGDB, or any IDE).  
4. Run the program and use the menu to manage customers.
